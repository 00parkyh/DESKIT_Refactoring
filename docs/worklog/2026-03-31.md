## 수정 이유
- `storage-minio`는 Docker 내부 호스트명이라 브라우저가 직접 접근할 수 없음
- 업로드 응답과 VOD URL 생성 시 내부 endpoint를 그대로 반환하면 브라우저에서 `ERR_NAME_NOT_RESOLVED`가 발생함
- public endpoint를 분리해서 브라우저에 공개 가능한 URL만 반환하도록 정리해야 함
