# 2026-06-14 작업 로그

## 작업 목적

- `live_chat` 테이블에 존재하지 않는 `is_hidden` 컬럼으로 인해 더미 데이터 입력이 실패하는 문제를 수정한다.

## 수정한 파일

- `src/main/resources/sql/dummy_data.sql`

## 수정 전 코드

```sql
INSERT INTO live_chat (..., is_world, is_hidden, vod_play_time)
VALUES (..., FALSE, FALSE, 5);
```

## 수정 후 코드

```sql
INSERT INTO live_chat (..., is_world, vod_play_time)
VALUES (..., FALSE, 5);
```

## 수정 이유

- `create_table.sql`의 `live_chat` 정의에는 `is_hidden` 컬럼이 없다.
- INSERT 컬럼과 값에서 `is_hidden` 항목을 함께 제거하여 현재 스키마와 일치시켰다.
